# docs/system — 系统内部知识库

面向开发、运维与系统侧 Agent 的技术文档，适合挂载为 **系统** Obsidian Vault。

| 目录 / 文件 | 内容 |
|-------------|------|
| `reference/` | API、配置、环境变量、架构、工具/技能参考（MkDocs「参考」） |
| `developer/` | 贡献指南、提交规范 |
| `design/` | 产品/架构设计草案与专项报告 |
| `api/` | 专题接口说明（如 Apps API） |
| `internal/` | 需求、技术长文、Bug 分析、第三方参考（**默认不进 MkDocs**） |
| `presentations/` | 演示稿（**默认不进 MkDocs**） |
| `proposals/` / `reports/` | 提案与报告占位 |
| `CHANGELOG-*.md` | 文档体系 / Knowledge Vault 变更记录 |
| `team-artifact-flow-sop.md` | 团队产物流转 SOP |

用户操作说明在 [`../user/`](../user/)。智能体值班产出在 [`../roles/`](../roles/)，不是本库的 SSOT。
