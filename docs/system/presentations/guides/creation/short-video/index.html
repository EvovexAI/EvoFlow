用 Plan 模式在本平台制作一支短视频

Supervisor 当制片：澄清需求 → 结构化计划 → 五工种串行协作 → 交付 MP4

需 Plan 协作 + 火山/Ark 媒体 Key 预计 30–60 分钟（含生成等待） 截图路径已预留

一、开始之前

确认环境就绪，并理解媒体工种如何分工

前置

你需要先具备什么

✓

已完成模型配置

主模型可用（见 配置模型与第一次对话）。复杂制片建议用推理能力较强的模型当 Supervisor。

🔑

媒体 API 凭据

生图/生视频走火山方舟（Seedream + Seedance）。在 设置 → 环境变量 配置 VOLCENGINE_API_KEY 或 ARK_API_KEY；或在 设置 → 模型 → 创意媒体 填写。

📁

工作区

绑定本地项目目录，产物默认写入 outputs/（剧本、关键帧、MP4、字幕成片）。

🧩

技能

启用 media-production、byted-ark-seedream-skill；Plan 视频流程可读 plan-video-production 技能说明。

和「普通聊天生视频」的区别：本教程走 Plan 协作——先出完整计划、你点「开始执行」后才跑脚本；各工种写在 plan 步骤里，而不是主会话随手 terminal。

工种（assigned_agent）产出物执行方式

二、跟着做一遍

从打开 Plan 到拿到成片的关键操作（细节可对照 Plan 协作流程）

步骤 1

启动 Plan 协作

  1. 打开 EvoFlow，绑定工作区（含 outputs/ 目录的项目文件夹）。
  2. 顶栏激活 任务规划 场景（Plan 协作）。
  3. 用自然语言描述目标，例如：「做一支 15 秒竖屏产品短片，科技风，主题：[你的产品]。」
  4. 也可使用快捷提示 「文生视频」 / 「配音字幕短片」 再改主题占位符。
步骤 2

澄清需求（侧栏问卷）

Supervisor 会追问：时长、画幅、风格、口播方向、参考素材等——回答越具体，计划越稳

建议确认项示例
主题 / 产品新款智能手表
时长15 秒 / 30 秒
画幅9:16 竖屏（抖音）或 16:9
视觉风格科技感 / 温暖 / 国风
口播功能三点 + slogan
步骤 3

查看结构化计划

计划定稿后应看到 5 个（或你要求的）步骤，每步指定媒体工种与验收标准

  • 点击 查看计划，检查步骤依赖是否：编剧 → 视觉 → 美术 → 导演 → 后期。
  • 每步应有 执行人、输出物路径、验收标准。
  • 不满意则在对话中说明修改意见，Supervisor 会重新生成计划。
快速单图/海报（不要分工）可走普通聊天 + terminal，不在本 Plan 流程内；本教程专注多工种短片。
步骤 4

点击「开始执行」

唯一正式授权入口 — 此后各工种按依赖串行跑脚本，主会话监工验收

  1. 输入框上方出现确认条：查看计划 · 修改计划 · 开始执行。
  2. 确认无误后点击 开始执行。
  3. 若之后改了计划内容，授权会撤销，需再次点击。
步骤 5

跟进执行与分步验收

  • 协作侧栏 / 工作流面板：看子任务状态、依赖、进度。
  • 每步完成后 Supervisor 对照验收标准检查 outputs/ 再解锁下一步。
  • 视频生成需等待异步任务 — 工种应跑 task_wait.py，不要未完成就交付。
  • 某步失败可修正、重试或改派（同一步最多重试 3 次）。

FAQ

常见问题

主会话自己跑 image_generate.py？

Plan 模式下应由 plan 步骤里的 media-artist 等工种执行，主会话负责监工与验收。

视频一直 generating

确认工种调用了 task_wait.py 等待 Ark 任务完成；不要跳过等待。

jimeng 失败换 wan/kling？

默认流水线不要自动换厂商；检查 Ark 接入点与 Key，同参数最多重试 1 次。

不要字幕可以吗？

可以。plan 里去掉 media-post 步骤，第 4 步 MP4 即为交付物。

和「配音字幕短片」快捷提示的关系？

快捷提示是起手话术；实际仍须走 Plan 澄清 → 计划 → 授权，并把工种写进步骤。

下一篇写什么？

可补充：单镜重跑、竖屏模板、多版本 A/B、或 Agnes/Kling 等其它 provider 专题。
