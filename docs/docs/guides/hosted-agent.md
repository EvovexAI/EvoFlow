# 托管智能体操作指南
---
## 1. 创建托管任务
### 方式一：界面按钮创建（推荐）
1. 在任意聊天页面，找到底部输入框旁边的「托管」按钮，点击
2. 页面右侧会弹出托管配置面板，填写配置：
   - **任务目标**：描述清楚要自动完成的事，越详细越好
   - **总轮次**：设置总共执行多少次，填0表示不限次数，一直运行到手动停止
   - **自动停止时间**（可选）：设置托管运行多久后自动停止，单位分钟，最长支持7*24小时（10080分钟）
   - **结果推送**：开启后任务执行结果、异常告警会自动推送到你配置的通知渠道
3. 点击「启动托管」按钮，托管任务就会在后台自动运行
### 方式二：聊天直接触发
直接在对话中输入任务要求，例如：
```
帮我创建一个托管任务，每小时检查一次服务器状态，运行7天后自动停止
```
发送后系统自动根据描述创建托管任务并开始运行
---
## 2. 托管运行说明
- 最长支持7*24小时不间断运行，达到全局最长运行时间会自动停止
- 执行过程中如果需要用户补充信息，会自动暂停等待反馈，等待时间最长7天，不会因为用户暂时离线被误杀
- 同时支持按次数、按时间两种停止规则：达到设定的总轮次、达到自动停止时间、任意一个条件满足就会自动停止
- 运行过程中产生的上下文会自动压缩，避免token溢出，最近10轮完整上下文会保留
---
## 3. 托管任务管理
托管运行过程中，右侧会一直展示托管运行面板：
- 实时展示当前运行状态、已执行轮次、最近执行结果
- 支持一键暂停/恢复运行，支持手动终止托管任务
- 运行过程中可以直接在聊天窗口发送补充指令，调整执行逻辑、修改目标，不需要重新创建托管
---
## 4. 历史运行记录
当前正在运行的托管任务可以在右侧面板查看实时运行日志，终止后的托管任务执行记录可以在聊天上下文查看完整的执行过程和最终结果
