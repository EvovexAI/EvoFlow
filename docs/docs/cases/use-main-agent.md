# 案例一：在对话中使用主智能体

## 场景说明

你已经安装并配置好了 EvoFlow，想要和默认的主智能体（Lead Agent）进行对话，让它帮你解答问题、分析内容或执行任务。

## 前置准备

- 已完成 [安装](../getting-started/installation.md)
- 已配置至少一个 AI 模型（API Key）
- 已启动 EvoPanel 并登录

## 操作步骤

### 步骤 1：打开对话页面

在 EvoPanel 左侧导航栏中，点击 **「对话」** 图标，进入对话页面。

![打开对话页面](../assets/cases/use-main-agent-step1.png)

> **截图说明**：展示 EvoPanel 左侧导航栏，标注"对话"图标的位置。

### 步骤 2：选择对话模式

在对话页面上方，你会看到四种模式选择：

| 模式 | 适用场景 |
|------|----------|
| **聊天** | 快速问答、简单咨询 |
| **规划** | 需要深度分析、多步骤思考的问题 |
| **执行** | 需要调用工具或委托子智能体完成的任务 |
| **无限** | 全部能力开放，适合复杂场景 |

选择一个适合你当前需求的模式。

![选择对话模式](../assets/cases/use-main-agent-step2.png)

> **截图说明**：展示对话模式选择区域，标注四种模式的切换按钮。

### 步骤 3：输入你的问题

在底部输入框中输入你的问题或指令，按回车发送。

**示例提示词：**

```
帮我分析一下目前 Python 项目中依赖管理的最佳实践，
有哪些主流方案？各自的优缺点是什么？
```

![输入问题](../assets/cases/use-main-agent-step3.png)

> **截图说明**：展示输入框输入问题的界面，标注发送按钮。

### 步骤 4：查看流式响应

主智能体会以流式方式输出回答，支持 Markdown 渲染。如果智能体调用了工具（如搜索、文件读取），你可以在对话中看到工具调用的过程。

![查看流式响应](../assets/cases/use-main-agent-step4.png)

> **截图说明**：展示智能体流式输出回答的界面，标注工具调用可视化区域。

### 步骤 5：多轮对话

你可以继续追问、要求补充或让智能体调整回答方向。主智能体会保持上下文连贯性。

**示例提示词：**

```
请帮我用表格的形式对比这几种方案，
并推荐一个最适合中型团队的选择。
```

![多轮对话](../assets/cases/use-main-agent-step5.png)

> **截图说明**：展示多轮对话的完整界面，标注上下文保持的效果。

### 步骤 6：上传文件或多模态输入（可选）

你可以粘贴截图或拖拽图片到对话中，主智能体会进行图片识别和分析。

![上传图片](../assets/cases/use-main-agent-step6.png)

> **截图说明**：展示上传图片后智能体识别和分析的界面。

## 小结

完成以上步骤后，你已经掌握了与主智能体对话的基本方式。主智能体是 EvoFlow 的核心交互入口，后续创建智能体、技能、定时任务等操作都可以在对话中通过自然语言指令完成。
