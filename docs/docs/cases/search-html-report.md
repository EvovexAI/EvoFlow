# 案例六：搜索汇总并输出分析 HTML

## 场景说明

你需要对某个主题进行深入研究，搜集多方信息，并生成一份结构化的分析报告。EvoFlow 的智能体可以自动完成搜索、筛选、汇总和生成 HTML 报告的全流程。

## 前置准备

- 已完成 [安装](../getting-started/installation.md)
- 已配置至少一个 AI 模型
- 已安装 `deep-research` 技能（参考 [案例四：安装技能](install-skill.md)）

## 操作步骤

### 步骤 1：打开对话页面

在 EvoPanel 左侧导航栏中，点击 **「对话」** 图标。

<!-- 截图占位：search-html-report-step1.png — 打开对话 -->

> **截图说明**：展示左侧导航栏，标注"对话"图标。

### 步骤 2：选择执行模式

在对话模式中选择 **「执行」** 或 **「无限」** 模式，因为深度搜索需要调用搜索工具和文件写入能力。

<!-- 截图占位：search-html-report-step2.png — 选择执行模式 -->

> **截图说明**：展示对话模式选择，标注"执行"模式。

### 步骤 3：输入研究指令

向智能体输入你的研究需求。指令越具体，搜索结果越精准。

**示例提示词：**

```
请帮我做一份关于"Rust 在 Web 后端开发中的应用现状"的深度研究报告。

要求：
1. 搜索近一年的技术资料、博客和论文
2. 对比主流 Web 框架（Actix-web、Axum、Rocket 等）
3. 分析性能数据和典型案例
4. 生成一份 HTML 格式的分析报告，包含图表和数据对比
```

<!-- 截图占位：search-html-report-step3.png — 输入研究指令 -->

> **截图说明**：展示输入框中输入完整研究指令的界面。

### 步骤 4：智能体自动搜索

智能体会自动调用搜索引擎，从多个来源获取信息。你会看到搜索过程的实时展示。

<!-- 截图占位：search-html-report-step4.png — 搜索过程 -->

> **截图说明**：展示智能体调用搜索工具、获取搜索结果的实时过程。

### 步骤 5：信息筛选与汇总

智能体会对搜索结果进行阅读、筛选和交叉验证，提取有价值的信息。

<!-- 截图占位：search-html-report-step5.png — 信息汇总 -->

> **截图说明**：展示智能体筛选和整理信息的对话过程。

### 步骤 6：生成 HTML 报告

智能体完成分析后，会生成一份 HTML 格式的报告文件，并在对话中展示预览或提供下载链接。

报告通常包含：
- 标题和摘要
- 核心技术对比（表格）
- 性能数据（图表）
- 典型案例
- 结论与建议

<!-- 截图占位：search-html-report-step6.png — HTML 报告 -->

> **截图说明**：展示生成的 HTML 报告预览或下载入口。

### 步骤 7：查看和下载报告

你可以在对话中找到报告的下载链接，或在项目的工作区目录中找到生成的 HTML 文件。

<!-- 截图占位：search-html-report-step7.png — 下载报告 -->

> **截图说明**：展示报告文件在工作区中的位置和下载方式。

### 步骤 8：进一步调整（可选）

如果对报告不满意，可以继续对话要求调整：

**示例提示词：**

```
请在报告中增加一个章节，对比 Rust 和 Go 在后端开发中的性能差异。
```

## 小结

深度搜索 + HTML 报告生成是 EvoFlow 智能体的强大能力之一。从信息搜集到结构化输出，全流程自动化，大幅提升研究效率。
