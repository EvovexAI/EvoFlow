## 推荐阅读顺序

如果你是第一次使用 EvoFlow，建议按以下顺序阅读：

1. **在对话中使用主智能体** → 先了解基本交互方式
2. **安装技能** → 快速扩展能力
3. **创建智能体** → 打造专属助手
4. **创建技能** → 进一步定制能力
5. **定时任务** → 让智能体自动工作
6. **搜索汇总并输出分析 HTML** → 进阶：研究分析场景
7. **Claude Code 持续对话** → 进阶：AI 辅助开发
8. **Supervisor 模式（超级总控智能体）** → 进阶：多智能体协同
