# Claude Code 操作指南
---
## 功能概述
Claude Code模式提供原生的Claude Code工具链能力，支持代码编辑、命令执行、多文件操作、项目架构设计等软件开发相关任务，适合各类编码、调试、重构场景。

---
## 启用与关闭
### 桌面端使用
在聊天窗口直接发送指令即可：
| 指令 | 功能说明 |
|------|----------|
| `/claude` 或 `/claude-code` | 启用当前会话的Claude Code直连模式 |
| `/lead` 或 `/main` | 关闭Claude Code模式，切回EvoFlow主智能体 |

启用后当前会话会自动使用Claude Code能力处理所有请求，无需额外配置。

---
## 飞书端使用
在绑定的飞书对话中同样可以使用Claude Code：
1. 发送 `/claude` 开启Claude Code专属模式
2. 直接发送代码需求，Claude Code会自动处理并回复
3. 发送 `/lead` 切回主智能体

---
## 常用功能特性
- **代码生成**：根据需求自动生成各类语言的代码片段和完整项目
- **代码调试**：自动分析错误日志，定位问题并提供修复方案
- **文件操作**：支持读取、写入、修改本地项目文件
- **命令执行**：可以直接运行终端命令，处理构建、测试、部署等任务
- **项目分析**：自动扫描项目结构，理解代码架构和依赖关系
- **重构优化**：提供代码重构建议，优化性能和可维护性

---
## 多会话协同
支持同时开启多个Claude Code会话并行处理不同任务，每个会话独立保存上下文，适合大型编码任务拆解为多个子任务并行处理，提升开发效率。
