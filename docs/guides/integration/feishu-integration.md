# 飞书渠道使用指南
---
## 功能概述
飞书渠道集成支持将EvoFlow的所有能力直接在飞书对话中使用，无需打开桌面端，支持群聊和单聊两种模式，包含所有核心功能：普通对话、智能体切换、目标任务、自动化、Claude Code模式等。

---
## 1. 飞书机器人配置
1. 打开飞书开放平台，创建企业自建应用，获取`App ID`和`App Secret`
2. 开启机器人能力，配置事件订阅和消息回调地址（按照EvoPanel设置页的提示填写）
3. 回到EvoPanel「设置」→「飞书配置」，填入`App ID`和`App Secret`，开启飞书渠道
4. 飞书开放平台发布应用，企业内部可用即可

---
## 2. 会话映射配置
可以将飞书的对话和EvoFlow的智能体绑定：
1. 进入EvoPanel「设置」→「飞书配置」→「会话映射」
2. 点击「添加映射」，选择要绑定的飞书群/用户
3. 选择对应的智能体，保存后该飞书对话的消息会自动转发给绑定的智能体处理
4. 也可以不绑定固定智能体，由用户通过指令动态切换

---
## 3. 飞书端常用指令
在飞书对话中直接发送指令即可：
| 指令 | 功能说明 |
|------|----------|
| `/new` | 新建独立会话线程，上下文和之前完全隔离 |
| `/claude` | 开启Claude Code专属模式，使用原生Claude Code能力 |
| `/claude [会话ID]` | 续接指定ID的历史Claude Code会话 |
| `/lead` 或 `/main` | 关闭Claude Code模式，切回EvoFlow主智能体 |
| `/agent [智能体名称]` | 切换到指定的智能体，例如`/agent 代码专家` |
| `/goal [任务目标]` | 直接创建并启动后台目标任务，例如`/goal 每小时检查服务器磁盘占用` |
| `/task [任务描述]` | 创建编排任务，智能体自动拆解成多步骤执行 |
| `/status` | 查看当前会话的任务状态、运行进度、目标任务列表 |
| `/stop` | 终止当前运行中的任务/目标 |
| `/help` | 查看所有可用指令说明 |
| `/models` | 查看可用模型列表 |
| `/agents` | 查看所有可用智能体列表 |
| `/memory` | 查看当前会话的记忆内容 |
| `/automation [操作]` | 自动化操作，支持创建、查看、暂停、删除自动化，别名`/cron`、`/schedule` |

---
## 4. 目标任务操作
- 创建目标：直接发送`/goal 任务目标描述`，自动创建并开始后台运行
- 查看状态：发送`/status`可以看到当前所有运行中目标任务的进度、已执行轮次、最近结果
- 停止目标：发送`/stop` 可终止当前目标任务
- 结果推送：目标任务的执行结果、异常告警会自动推送到飞书对话中，每一步执行进度都会实时更新
- 人工反馈：如果目标任务需要用户补充信息，会自动暂停并@相关用户等待反馈，收到反馈后自动继续执行

---
## 5. 自动化操作
通过`/automation`指令可以直接在飞书中管理自动化：
```
# 创建自动化
/automation create 每天早上9点生成昨日业务数据报表

# 查看所有自动化
/automation list

# 暂停自动化
/automation pause 任务ID

# 恢复自动化
/automation resume 任务ID

# 删除自动化
/automation delete 任务ID

# 查看执行历史
/automation history 任务ID
```

---
## 6. 卡片式交互体验
飞书端所有回复均采用交互式卡片格式：
- 运行中的任务会动态更新卡片内容，实时展示进度
- 代码块、列表、链接等内容会自动美化渲染
- 长内容自动折叠，支持展开查看完整内容
- 异常错误会用红色卡片标识，方便快速识别问题
