# Skills 操作指南
---
## 页面入口
点击左侧菜单栏「扩展」分类下的「Skills」进入技能管理页面，包含两个Tab：「已安装」和「搜索安装」。
---
## 管理已安装技能
进入「已安装」Tab可以查看所有已经安装的技能：
- 每个技能卡片展示图标、名称、功能描述、分类
- 卡片右上角的开关可以快速启用/禁用对应技能，禁用后所有Agent都无法调用该技能
- 非公共分类的自定义技能可以点击卡片右下角的删除按钮删除
- 顶部搜索框支持按技能名称/描述过滤
- 右侧状态下拉可以筛选查看「全部」「已启用」「已禁用」技能
---
## 安装新技能
### 方式一：从技能市场安装
1. 切换到「搜索安装」Tab，自动加载热门技能
2. 可以在顶部搜索框输入关键词搜索需要的技能
3. 找到需要的技能后，点击卡片右下角的「安装」按钮，自动完成安装
4. 安装完成后按钮变为「已安装 ✓」，技能自动启用
### 方式二：导入本地/自定义技能
1. 点击页面右上角的「导入本地技能」按钮
2. 填写配置：
   - 技能名称：唯一标识（英文，无空格）
   - 技能路径：支持本地目录路径、Git仓库地址、npm包名
   - 导入后立即启用：默认勾选
3. 点击确认即可完成导入
---
## 给Agent配置技能权限
安装后的技能需要在Agent配置中启用才能使用：
1. 进入左侧「Agent管理」页面，编辑对应Agent
2. 切换到「技能模块」Tab
3. 勾选需要让该Agent调用的技能，或者勾选「全部可用」使用所有已启用技能
4. 保存即可生效，后续该Agent对话时会根据需求自动调用对应技能
