# 面板设置

EvoPanel 左下角「设置」按钮（或快捷键唤起）打开**设置弹窗**，集中管理应用偏好、模型快速切换、环境变量、IM 凭证、记忆、快捷键与版本信息。所有变更**自动持久化**到 `panel-settings`，并按实例（本机 / Docker / 远程）隔离。

> 设置弹窗与左侧主菜单的「模型」「消息渠道」「记忆」存在功能互补：**主菜单偏精细管理**（全屏、批量），**弹窗偏快速调整**（窄屏、单条）。

---

## 弹窗结构

设置弹窗采用「左侧导航 + 右侧多面板」布局，共 7 个 Tab：

| Tab | 用途 |
|-----|------|
| **通用** | 主题 / 字体 / 语言 / 工具审批 / 数据目录等基础偏好 |
| **模型** | 默认对话模型快速切换、连接状态查看 |
| **环境变量** | 为 terminal / 技能脚本注入 `KEY=VALUE` |
| **IM 通信** | 飞书 / Slack / Telegram 等渠道凭证 |
| **记忆** | 全局记忆与各 Agent 记忆的快速浏览与编辑 |
| **快捷键** | 自定义聊天与全局快捷键 |
| **关于** | 版本号、更新检查、许可证与致谢 |

每个 Tab **按需懒加载**，切换时不重渲染容器，避免高度抖动。URL 同步 hash（`#/settings?tab=xxx`），可直接外链到具体 Tab。

详细每个 Tab 的能力要点见 [EvoPanel 桌面端使用指南](./evopanel-guide.md#二设置弹窗) 的速查表。

## 常用偏好速查

### 主题与外观（通用 Tab）

| 项 | 默认 | 说明 |
|----|------|------|
| 主题模式 | 跟随系统 | 日间 / 夜间 / 跟随系统 |
| 字体大小 | 中 | 小 / 中 / 大 / 超大 |
| 界面语言 | 跟随系统 | 中文 / 英文；AI 回复语言默认跟随界面 |

### 工具审批策略（通用 Tab）

控制 EvoFlow 主智能体调用高风险工具时是否需要用户确认：

| 策略 | 行为 |
|------|------|
| **默认询问** | 终端 / 删文件 / 网络请求等弹出审批条；用户点同意才执行 |
| **自动批准** | 全部直接执行（**仅推荐在本地受信任环境使用**） |
| **仅会话内授权** | 同会话首次审批后，后续同名工具直接通过 |

也可以对单个工具（如 `terminal`、`delete`）单独覆盖策略。

### 数据目录（通用 Tab）

| 目录 | 用途 |
|------|------|
| `~/.evoflow/` | 主数据目录（会话、记忆、自动化任务、配置） |
| `~/.evoflow/tasks/automations/` | 自动化 TOML 配置 |
| `/mnt/user-data/` | 虚拟沙箱路径（`uploads/` 与 `outputs/`） |

通用 Tab 提供「打开数据目录」「清理缓存」入口。

### 模型快速切换（模型 Tab）

只展示**对话主模型**的下拉与状态，不含创意媒体模型。若要添加厂商、配置温度/思考预算等参数，请去主菜单「模型配置」。

### 环境变量（环境变量 Tab）

以 `KEY=VALUE` 形式集中维护，写入后对沙箱内的 `terminal`、技能脚本、子代理统一可见，**不需重启**：

```
OPENAI_API_KEY=sk-...
HTTP_PROXY=http://127.0.0.1:7890
GIT_COMMITTER_NAME=Alice
```

敏感值在列表中以掩码（`sk-•••••`）显示。

### 快捷键（快捷键 Tab）

预置常用动作（发送、新会话、停止生成、打开命令面板、切换 Agent 等），冲突检测会在与系统级快捷键重叠时给出告警。一键恢复默认。

## 多实例切换

EvoPanel 支持同时管理**本机**、**Docker**、**远程**三类实例：

- 设置弹窗内的设置**按实例独立存储**——本机的快捷键与远程的快捷键互不影响
- 左侧菜单顶部的实例切换器可在任意时刻切换当前实例
- 远程实例添加方式见 [EvoPanel 桌面端使用指南](./evopanel-guide.md)
