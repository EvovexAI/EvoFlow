# 智能体（角色）操作指南
---
## 1. 快速创建智能体（最简单方式，对话直接触发）
直接在聊天窗口输入你的角色要求，例如：
```
帮我创建一个专门写Python代码的智能体，要求代码风格简洁、注释详细、擅长处理数据脚本
```
发送后系统会自动创建符合要求的智能体，提示创建完成即可直接使用。

---
## 2. 手动创建/精细配置智能体
1. 点击左侧菜单「智能体」进入管理页面
2. 点击「新建智能体」按钮
3. 按需求填写配置：
   - **基础信息**：设置智能体名称、头像、功能描述
   - **Soul设定**：填写角色人设、回复风格、专业领域、行为规则等核心prompt，决定智能体的定位和回复方式
   - **能力配置**：
     - 「技能权限」：勾选这个角色可以使用的技能，没勾选的技能不会被调用
     - 「工具权限」：勾选允许使用的工具（终端执行、文件读写、网络请求等），精细控制角色能力范围
     - 「MCP服务权限」：勾选允许该角色调用的MCP服务，控制访问外部系统/服务的权限
4. 点击「保存」完成创建

---
## 3. 切换使用智能体
两种切换方式：
1. 聊天页面顶部下拉选择框，直接点击要使用的智能体即可切换
2. 聊天中直接发送指令：`切换到[智能体名称]`，自动快速切换

---
## 4. 管理已有智能体
进入左侧「智能体」页面：
- 点击智能体卡片的「编辑」按钮，可以修改角色设定、模型、权限等配置
- 点击「删除」按钮，可以移除不需要的智能体
