# 任务中心操作指南
---
## 概述
任务中心是**复杂多任务协作的核心功能**，适用于需要多步骤、多智能体协作完成的大型任务。AI会自动将复杂任务拆解为多个子任务，协调不同的智能体并行/串行执行，全程监控进度，最终汇总完成整个大任务。

---
## 一、任务中心入口
点击左侧菜单栏的「任务中心」进入，页面展示所有任务的卡片列表，每个卡片显示任务名称、描述、运行状态、进度、子任务完成情况等信息。

---
## 二、任务列表操作
### 1. 新建任务
点击页面右上角的「新建任务」按钮，填写：
- **任务名称**：简短描述任务目标，比如「竞品分析报告」、「Python管理系统开发」
- **任务描述**：详细说明任务的具体要求、输出格式、截止时间等信息，越详细AI拆解越准确
点击确定后，系统会自动创建任务，Supervisor智能体开始分析任务并拆分子任务。

### 2. 任务状态说明
| 状态 | 说明 |
|------|------|
| 待开始 | 任务已创建，还未开始规划 |
| 规划中 | Supervisor正在分析任务，拆分子任务，制定执行计划 |
| 执行中 | 任务正在执行，子任务在并行/串行运行 |
| 已暂停 | 任务被手动暂停，可以恢复执行 |
| 已完成 | 所有子任务执行完成，任务结束 |
| 失败 | 任务执行过程中出现错误，无法继续 |
| 已取消 | 任务被手动取消 |

### 3. 单个任务操作
每个任务卡片有以下操作按钮：
- **查看详情**：进入任务详情页面，查看子任务进度、执行结果、对话记录
- **💬 实时对话**：跳转到聊天页面，查看任务执行的实时流式输出，可以直接和执行任务的智能体交互
- **开始执行**：当任务处于「规划中」状态时，点击确认执行计划，开始运行任务
- **暂停**：当任务处于「执行中」状态时，点击暂停任务运行
- **删除**：删除任务及其所有子任务，操作不可恢复

### 4. 批量操作
点击「批量选择」按钮进入批量操作模式：
- 可以勾选多个任务，支持批量启动、批量暂停、批量删除操作
- 适合同时管理大量任务的场景
- 点击「取消选择」退出批量模式

### 5. 搜索任务
在顶部搜索框输入关键词，可以按任务名称、描述、状态搜索过滤任务列表。

---
## 三、任务详情页面
点击任务卡片的「查看详情」按钮进入任务详情页，页面分为几个核心区域：

### 1. 顶部操作区
- 显示任务名称、描述、当前状态、整体进度条
- 操作按钮：
  - **开始/暂停/重新开始**：控制任务的运行状态
  - **添加子任务**：手动新增子任务到当前任务中
  - **📋 任务计划**：查看AI生成的完整任务执行计划文档
  - **💬 查看实时对话**：跳转到聊天页面查看实时执行过程
  - **刷新**：刷新页面状态

### 2. 子任务列表区
展示当前任务拆解出的所有子任务，每个子任务卡片显示：
- 子任务名称、分配的智能体、运行状态
- 进度条、执行耗时
- 操作按钮：查看详情、暂停/继续、重新执行、查看结果
- 子任务状态和主任务状态一致，支持单独控制子任务运行

### 3. 对话记录区
右侧面板展示Lead Agent的完整对话记录，包括：
- 任务拆解过程
- 子任务分配和调度记录
- 智能体之间的协作沟通
- 执行过程中的工具调用日志
- 错误信息和处理过程
执行过程中会自动刷新，可以看到实时输出。

---
## 四、任务执行流程
### 1. 创建任务
填写任务名称和描述，系统自动创建任务。

### 2. 规划阶段
Supervisor智能体自动分析任务，拆解为多个子任务，制定执行顺序、分配对应的智能体，生成完整的执行计划。规划完成后可以点击「查看任务计划」查看完整的拆解文档。

### 3. 确认执行
确认计划无误后，点击「开始执行」按钮，任务进入执行阶段。如果对计划不满意，可以直接在聊天页面和Supervisor沟通修改，或者手动添加/删除子任务。

### 4. 执行阶段
系统按照计划自动执行子任务：
- 依赖的子任务会串行执行，无依赖的子任务会并行执行，提高效率
- 每个子任务分配给对应的专业智能体执行（比如代码任务分配给代码专家，写作任务分配给文案专家）
- 执行过程中出现错误会自动重试，重试失败会暂停任务通知人工处理
- 可以随时暂停任务，修改子任务要求或者调整计划，然后继续执行

### 5. 完成阶段
所有子任务执行完成后，Supervisor会汇总所有子任务的结果，生成最终的完整输出，任务标记为「已完成」。

---
## 五、高级功能
### 1. 人工干预
任务执行过程中可以随时介入：
- 直接在聊天页面给智能体发消息，修改任务要求
- 暂停任务，手动修改子任务的内容、分配的智能体
- 终止执行失败的子任务，重新执行或者替换为其他子任务
- 手动添加新的子任务到执行队列中

### 2. 跨会话关联
每个任务会自动关联对应的聊天会话，点击「查看实时对话」可以直接跳转到对应的聊天页面，和执行任务的智能体实时交互，不需要手动查找会话。

### 3. 任务导出
执行完成的任务可以导出完整的执行过程文档，包括所有子任务的结果、对话记录、工具调用日志等。

### 4. 任务模板
常用的任务可以保存为模板，下次创建相同类型任务时直接使用模板，不需要重复填写描述。
