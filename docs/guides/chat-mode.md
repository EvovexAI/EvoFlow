# 会话模式
---
## 对话功能
- **多模态输入**：支持发送文本、图片、文件，支持视觉的模型会自动识别图片内容
- **流式输出**：回复实时逐字渲染，不需要等待全部生成完成即可阅读
- **工具调用可视化**：AI调用工具、执行代码的过程会实时展示，可展开查看完整运行日志和返回结果
- **会话管理**：左侧侧边栏可查看所有历史会话，支持搜索、删除会话，点击即可切换上下文
- **线程模式**：支持在同一个会话中创建多个独立线程，适合同时处理多个相关但隔离的子任务

---
## 文件上传与解析
点击聊天框右侧上传按钮，或直接拖拽文件到聊天窗口即可上传：
- **文档类**：支持PDF/Word/Excel/PPT/markdown/txt等格式，自动解析文本内容
- **代码类**：支持所有常见编程语言的代码文件、压缩包，甚至整个项目文件夹，AI可以直接读取分析代码结构和内容
- **图片类**：支持JPG/PNG/GIF等常见图片格式，视觉模型可以理解图片内容，支持截图直接粘贴上传
- 上传后AI会自动识别文件内容，可直接针对文件内容提问、要求分析、修改、重构等操作
