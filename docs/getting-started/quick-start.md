# 5分钟快速上手
---
## 1. 安装
[Releases页](https://github.com/EvovexAI/EvoFlow/releases) 下载对应系统安装包直接安装即可。

---
## 2. 配置模型
1. 打开EvoPanel，进入「设置」→「模型管理」，点击「添加模型」
2. 「厂商」下拉选你要用的AI服务商，「模型名称」选对应型号
3. 填入你从服务商官网获取的API密钥
4. 点击「测试连接」成功后，勾选「设为默认模型」保存即可

---
## 3. 开始使用
回到聊天页，直接输入需求发送即可，AI会自动调用工具完成任务。

---
## 特色功能快速上手
### 托管任务（后台自动执行）
- 方式1：右上角「+」→「新建托管」，输入任务目标点击启动
- 方式2：聊天中直接说：`帮我创建托管任务，每10分钟监控xxx网站是否可访问`
- 查看/管理：「任务中心」→「托管任务」

### Plan模式（自动触发，先确认方案再执行）
当你发送复杂多步任务需求时，系统会自动触发Plan模式：
1. AI会先输出完整的执行方案，包含任务拆解、步骤安排、预期结果
2. 你确认方案没问题，回复「确认执行」，AI就会按照方案分步执行
3. 如果方案有需要调整的地方，直接告诉AI修改即可，调整完再确认执行

### 创建专属智能体
聊天中直接说：`帮我创建一个专门写前端的智能体，精通Vue3，代码注释完整`，自动创建完成即可使用。
