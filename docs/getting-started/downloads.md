# 下载

**桌面客户端** 的安装包与 EvoFlow 本体一并发布在 **[EvovexAI/EvoFlow Releases](https://github.com/EvovexAI/EvoFlow/releases)**。源码位于本仓库 [`evopanel/`](https://github.com/EvovexAI/EvoFlow/tree/main/evopanel)。

安装后打开应用，在 **设置 → 模型** 中配置 API 密钥即可开始使用；详细界面说明见 [桌面端使用指南](../guides/configuration/evopanel-guide.md)。若需从源码部署后端，请参考 [安装说明](installation.md)。
