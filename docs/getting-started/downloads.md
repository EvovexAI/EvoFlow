# 下载

**桌面客户端（EvoPanel）** 的安装包与 EvoFlow 本体一并发布在 **[EvovexAI/EvoFlow Releases](https://github.com/EvovexAI/EvoFlow/releases)**。源码位于本仓库 [`evopanel/`](https://github.com/EvovexAI/EvoFlow/tree/main/evopanel)。

安装后请按 [安装说明](installation.md) 准备 `config.yaml`、API 密钥并启动后端；使用说明见 [EvoPanel 使用指南](../guides/configuration/evopanel-guide.md)。
