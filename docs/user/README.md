# docs/user — 用户指南

面向**终端用户**的操作说明和概念解释。产品内置知识库与此目录保持同步，供 AI 助手和用户查阅。

| 目录 | 内容 |
|------|------|
| `getting-started/` | **产品总览**、介绍、下载、安装、快速上手 |
| `tutorials/` | 手把手教程 |
| `guides/` | 按场景的操作指南（对话、任务、配置、安全等） |
| `cases/` | 最佳实践与用法案例 |
| `explanation/` | 概念说明（为何这样设计、Agent/技能/记忆等） |
| `meta/` | 术语表 |

**推荐阅读顺序**：[产品总览](getting-started/product-overview.md)（给谁用 · 解决什么 · 功能怎么串）→ 安装上手 → 按需打开 `guides/`。

> 内部架构文档见 `docs/system/`，不要将需求稿、接口字段说明、内部设计文档放入本库。
