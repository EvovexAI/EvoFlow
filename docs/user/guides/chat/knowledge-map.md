# 思维导图

> **比如 AI 正在跑一个长任务，你想知道它卡在哪了？**
>
> 打开右侧思维导图面板，你会看到：
> - **当前目标**：比如"调研 3 家 AI Agent 平台"
> - **已查的分支**：LangGraph 已查完 ✅，CrewAI 正在查 🔄，AutoGen 待查 ⏳
> - **卡在哪**：某个节点显示"找不到定价信息，尝试官网搜索"——原来卡在数据源了
> - **结论**：已查完的分支有结论，没用的分支已排除
>
> 思维导图跟聊天记录互补：聊天看"说了什么"，导图看"在想什么、追哪条线"。
> **功能关系**：思维导图是聊天消息流的补充视图——聊天看"Agent 怎么说"，导图看"Agent 心里怎么想"；在[Plan 模式](plan-mode.md)和[目标智能体](goal-agent.md)执行长任务时维护最勤，是排障和回放的好帮手。

EvoFlow 在聊天页面右侧提供一块「**思维导图**」面板——它是 Agent 边干活边自己整理出来的**思路图**。你不一定要看，但需要的时候点开它，能一眼看到 Agent 正在追什么目标、想到了哪一步、卡在哪里。本文讲：它是什么、怎么打开关掉、面板里几个按钮分别干嘛。

---

## 一、它是什么

把它想成 Agent 的**便签墙**：

- 工作开始时定一个**总目标**（goal）
- 干活的过程中，每一步新发现的事实、问题、初步结论，会被写成一张张**便签**（节点）
- 便签之间还会画**连线**（边）：A 依赖 B、A 推翻 C、A 是 B 的结论
- 任务结束或某个分支查清楚后，便签会被标记 `已验证 / 已解决 / 被推翻`

> 它和聊天消息流是**两套视图**：聊天看"Agent 怎么说"，思维导图看"Agent 心里怎么想"。

---

## 二、怎么打开和关掉

聊天页右上角工具栏有一个切换按钮：

| 当前状态 | 按钮提示 | 点一下会 |
|----------|---------|---------|
| 关 | **显示思维导图** | 右侧弹出思维导图侧栏 |
| 开 | **隐藏思维导图** | 收起侧栏 |

或者侧栏右上角点 **×** 也能关掉。

> 关掉**不会**让 Agent 停止维护这张图——下次打开还在，只是面板被隐藏。

---

## 三、面板里有什么

打开侧栏后，从上到下你会看到：

- **总目标**（顶部一句话）——Agent 当前会话的核心目的
- **摘要**（次顶部）——把目标拆成几句话的当前进度
- **节点树**——按层级展开的便签，每张便签可能是：
  - **flow:** 一条调查分支（"我在查 XX"）
  - **file:** 涉及的某个文件
  - **gap:** 待验证的假设
  - **claim:** 已得出的结论
- **每张便签的颜色 / 角标**表示状态：
  - 灰色 = 进行中
  - 绿色 = 已验证 / 已解决
  - 红色 = 已推翻
  - 黄色 = 被外部因素阻塞

---

## 四、面板顶部的小按钮

侧栏标题右边一排小图标：

| 图标 | 功能 |
|------|------|
| **视图切换** | 切换树视图 / 图视图（节点+连线） |
| **刷新** | 立即拉取最新的图，平时也会自动更新 |
| **展开** | 把侧栏内容铺到一个大弹窗里，看大图时用 |
| **×** | 关闭侧栏 |

---

## 五、什么时候适合打开它

- 看 Agent **跑长任务**时（目标、Plan、调研），想知道它到底关注了什么
- Agent **回复很久没动静**，怀疑卡哪了 → 看导图能看到它在哪个分支
- 任务结束后想**回看**——这张图比聊天记录更结构化
- 排障：看 Agent 是否漏了某个关键事实

普通对话、简单问答用不上，关闭即可。

---

## 六、常见问题

**Q：面板是空的，写着"暂无思维导图节点"？**
说明 Agent 当前会话还没开始维护这张图——通常发生在纯闲聊、不涉及代码或排障的场景。开始让它干活后图自然出现。

**Q：能不能让 Agent 别画这张图，省点 Token？**
当前没有专门的开关，但纯闲聊场景下 Agent 不会主动维护它。如果你介意，可以告诉它"本次对话不需要维护知识图谱"。

**Q：这张图能导出吗？**
当前侧栏只看不导出。如果你要带走，截图或直接复制节点文字即可。

**Q：我点了关闭，下次打开还在吗？**
在的——图本身存在会话状态里，关掉只是隐藏侧栏。新建一个会话才会从空白开始。
