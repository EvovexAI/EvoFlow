# 案例三：创建技能

> **比如你想让 AI 能"自动生成 PPT"——但系统里没有这个技能**：
>
> 1. 创建一个文件夹，里面放一个 `SKILL.md` 文件
> 2. 在 `SKILL.md` 里写清楚技能是做什么的、怎么用
> 3. 可以配一些脚本（scripts/）、参考资料（references/）、素材（assets/）
> 4. 保存后 AI 就能调用这个技能了
>
> 技能是 AI 的能力扩展模块。创建技能前建议先了解智能体的基本概念（案例二），创建好的技能可以安装到角色上使用。

## 场景说明

技能（Skill）是扩展智能体能力的模块化组件。当你发现智能体缺少某个专业能力时，可以为它创建一个技能。比如：自动生成 PPT、调用特定 API、处理特定格式的文件等。

## 前置准备

- 已完成[安装](../getting-started/installation.md)
- 了解技能的基本概念
- 知道你想让智能体学会什么新能力

## 操作步骤

### 步骤 1：理解技能结构

一个技能是一个文件夹，包含以下结构：

```
my-skill/
├── SKILL.md          # 技能主文件（必需）
├── scripts/          # 可执行脚本（可选）
├── references/       # 参考资料（可选）
└── assets/           # 模板、图标等资源（可选）
```

其中 `SKILL.md` 是最核心的文件，包含 YAML 元信息和技能描述。
