# 案例二：创建智能体

> **比如你经常做数据分析，想要一个"Python 数据分析师"专属角色**：
>
> 在聊天里说："帮我创建一个专门写 Python 数据分析脚本的智能体，偏好 Pandas，注释详细，默认用 DeepSeek V4"
>
> AI 自动生成这个角色的 SOUL（人设）、系统提示词、工具白名单，保存即用。
>
> 创建好的角色可以作为 Supervisor 模式（案例八）的子智能体被调用，也可以配合安装的技能（案例四）扩展能力。

## 场景说明

主智能体适合通用任务，但如果你有特定领域的需求（如专门做数据分析、内容审核、代码审查），可以创建一个具有专属身份和能力的自定义智能体。

## 前置准备

- 已完成[安装](../getting-started/installation.md)
- 已配置至少一个 AI 模型
- 了解 SOUL.md 的基本概念（智能体的"人格"定义）

## 操作步骤

### 步骤 1：进入角色管理

在 EvoFlow 左侧导航栏中，点击 **「角色」** 图标，进入智能体管理页面。
