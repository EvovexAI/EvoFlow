# 案例一：在对话中使用主智能体

> **比如你刚装好 EvoFlow，想先试试跟它聊聊天**：
>
> - 在输入框打"帮我查一下今天 AI 圈有什么新闻"——AI 直接回答
> - 打"用 Python 写一个快速排序"——AI 直接写代码
> - 打"帮我解释一下什么是 RESTful API"——AI 给你解释
>
> 这就是主智能体，EvoFlow 的默认入口。就像跟 ChatGPT 聊天一样，但后续可以升级到更复杂的能力。
>
> 这是所有案例的基础——先学会跟主智能体对话，再学后面的创建角色、装技能、自动化等。

## 场景说明

你已经安装并配置好了 EvoFlow，想要和默认的主智能体（Lead Agent）进行对话，让它帮你解答问题、分析内容或执行任务。

## 前置准备

- 已完成[安装](../getting-started/installation.md)
- 已配置至少一个 AI 模型（API Key）
- 已启动 EvoFlow 并登录

## 操作步骤

### 步骤 1：打开对话页面

在 EvoFlow 左侧导航栏中，点击 **「对话」** 图标，进入对话页面。
