# 案例四：安装技能

> **比如你想让 AI 能做"深度研究"——联网搜索、多来源对比、输出结构化报告**：
>
> 1. 打开技能市场，找到 `deep-research` 技能
> 2. 一键安装
> 3. 在角色管理里给 AI 勾上这个技能
> 4. 现在 AI 就能做深度研究了
>
> 安装技能是扩展 AI 能力最快的方式。如果你需要市场上没有的技能，可以自己创建（案例三）。

## 场景说明

EvoFlow 提供了丰富的预置技能，你也可以从技能市场或社区安装别人创建的技能，快速扩展智能体的能力。

## 前置准备

- 已完成[安装](../getting-started/installation.md)
- 已启动 EvoFlow 并登录

## 操作步骤

### 步骤 1：进入技能管理

在 EvoFlow 左侧导航栏中，点击 **「技能」** 图标，进入技能管理页面。
