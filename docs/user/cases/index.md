# 最佳实践案例

> **本系列围绕两大生产力工具展开**：
>
> **智能体员工**——把 AI 从"你问它答"变成"主动上班干活"。配岗位、定职责、设审批，它自己巡检、审查、写报告，你只用在待审批 Tab 拍板。
>
> **应用中心**——把跑通的工作流固化下来，以后填参数就能再跑。不用每次从零跟 AI 说一遍。
>
> 建议按顺序阅读，从雇佣第一个员工到编排完整交付流水线。
