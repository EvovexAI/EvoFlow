# 最佳实践案例

> **想找某个功能怎么用？按顺序看就行**：
>
> - **案例一**：跟主智能体聊天——基础中的基础
> - **案例二**：创建专属角色——比如"Python 数据分析师"
> - **案例三**：自己写个技能——让 AI 能做系统没有的能力
> - **案例四**：安装技能——从市场装现成的
> - **案例五**：自动化——让 AI 定时干活
> - **案例六**：搜索出报告——深度研究并输出 HTML
> - **案例七**：Claude Code——大段编码
> - **案例八**：Supervisor 模式——多 AI 角色协作
>
> 每个案例可独立阅读，但按顺序学效果更好。

本系列文档通过真实场景案例，展示 EvoFlow 各项功能的最佳实践。每个案例都包含分步操作指南和用户视角的提示词示例，方便你快速上手。

## 推荐阅读顺序

如果你是第一次使用 EvoFlow，建议按以下顺序阅读：

1. **在对话中使用主智能体** → 先了解基本交互方式
2. **安装技能** → 快速扩展能力
3. **创建智能体** → 打造专属助手
4. **创建技能** → 进一步定制能力
5. **自动化** → 让智能体自动工作
6. **搜索汇总并输出分析 HTML** → 进阶：研究分析场景
7. **Claude Code 持续对话** → 进阶：AI 辅助开发
8. **Supervisor 模式（超级总控智能体）** → 进阶：多智能体协同
