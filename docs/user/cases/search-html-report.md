# 案例六：搜索汇总并输出分析 HTML

> **比如你想研究"2025 年 AI  Agent 发展趋势"，出一份带图表的 HTML 报告**：
>
> 1. 先装好 `deep-research` 技能（案例四）
> 2. 切换 Agent 模式，输入"帮我深度研究这个主题，输出 HTML 报告"
> 3. AI 自动搜索→筛选→汇总→生成结构化的 HTML 分析报告
>
> 结合自动化（案例五）可以定时生成报告，比如每周自动出一份竞品动态报告。

## 场景说明

你需要对某个主题进行深入研究，搜集多方信息，并生成一份结构化的分析报告。EvoFlow 的智能体可以自动完成搜索、筛选、汇总和生成 HTML 报告的全流程。

## 前置准备

- 已完成[安装](../getting-started/installation.md)
- 已配置至少一个 AI 模型
- 已安装 `deep-research` 技能（参考[案例四：安装技能](install-skill.md)）

## 操作步骤

### 步骤 1：打开对话页面

在 EvoFlow 左侧导航栏中，点击 **「对话」** 图标。
