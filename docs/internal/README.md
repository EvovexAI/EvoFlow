# 内部资料（不参与文档站构建）

本目录存放**需求分析、详细设计、验收报告、历史技术长文、第三方参考资料原文**等，供贡献者在仓库内查阅。

- 这些内容**不会**被 MkDocs 构建进官方站点（见仓库根目录 `mkdocs.yml` 中的 `exclude_docs`）。
- 对外用户请使用站点导航中的 [快速开始](../getting-started/introduction.md)、[操作指南](../guides/evopanel-guide.md)、[参考手册](../reference/api-reference.md) 等页面。
- 若某篇内部文档中的流程与当前代码不一致，**以代码与 `config.example.yaml` 为准**。

子目录说明：

| 目录 | 内容 |
|------|------|
| `requirements/` | 原 `docs/需求文档` |
| `technical/` | 原 `docs/技术文档` |
| `third-party-references/` | 原 `docs/references`（第三方提示词等，非 EvoFlow 行为说明） |
