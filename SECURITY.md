# Security Policy

## Supported Versions

As evo-flow doesn't provide an official release yet, please use the latest version for the security updates.
Currently, we have two branches to maintain:
* main branch for evo-flow 2.x
* main-1.x branch for evo-flow 1.x 

## Reporting a Vulnerability

Please go to https://github.com/bytedance/evo-flow/security to report the vulnerability you find.
